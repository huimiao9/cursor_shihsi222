# 📖 网页版日记本

一个功能完整、界面美观的网页版日记本应用，使用纯前端技术开发，数据保存在浏览器本地存储中。

## ✨ 功能特性

- 📝 **创建日记** - 记录标题、日期、心情和内容
- ✏️ **编辑日记** - 随时修改已有的日记内容
- 🗑️ **删除日记** - 删除不需要的日记（带确认提示）
- 🔍 **搜索功能** - 按标题或内容搜索日记
- 😊 **心情筛选** - 按心情emoji筛选日记
- 📅 **日期排序** - 按日期升序或降序排列
- 📊 **统计信息** - 显示总日记数和总字数
- 💾 **本地存储** - 数据保存在浏览器localStorage中
- 📱 **响应式设计** - 完美适配手机、平板和电脑
- 🎨 **现代化UI** - 渐变背景、平滑动画、优雅设计

## 🚀 快速开始

### 方法一：直接打开

1. 下载或克隆此项目
2. 直接在浏览器中打开 `index.html` 文件
3. 开始使用！

### 方法二：使用本地服务器

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 🎯 使用说明

### 创建日记

1. 在页面顶部的编辑器中填写：
   - 标题：给你的日记起个名字
   - 日期：选择日记日期（默认为今天）
   - 心情：选择当天的心情emoji
   - 内容：写下你的想法和故事
2. 点击"保存日记"按钮

### 编辑日记

1. 在日记列表中找到要编辑的日记
2. 点击右上角的 ✏️ 图标
3. 修改内容后点击"更新日记"

### 删除日记

1. 在日记列表中找到要删除的日记
2. 点击右上角的 🗑️ 图标
3. 在确认对话框中点击"删除"

### 搜索和筛选

- **搜索**：在搜索框中输入关键词，会实时过滤标题和内容
- **心情筛选**：在下拉菜单中选择特定心情
- **排序**：选择"最新优先"或"最早优先"

## 💾 数据存储

所有日记数据都保存在浏览器的 `localStorage` 中，数据结构如下：

```javascript
{
  id: "唯一标识符",
  title: "日记标题",
  date: "2025-10-23",
  mood: "😊",
  content: "日记内容...",
  createdAt: "创建时间戳",
  updatedAt: "更新时间戳"
}
```

**注意**：
- 数据只存储在本地浏览器中
- 清除浏览器数据会删除所有日记
- 不同浏览器的数据不会同步
- 建议定期导出备份（可以通过浏览器开发者工具）
